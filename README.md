 # Vitaris II Launcher

Launcher oficial del servidor de Minecraft **Vitaris II**.

Gestiona automáticamente mods, configuraciones y actualizaciones del servidor.

## Requisitos

- Node.js 20.x
- npm

## Instalación (desarrollo)

```bash
npm install
npm start
```

## Build

```bash
# Windows
npm run dist:win

# macOS
npm run dist:mac

# Linux
npm run dist:linux
```
